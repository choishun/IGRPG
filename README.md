# IGRPG - IGRUS 동방을 지켜라!
## 소개
IGRUS 동아리 신입 부원이 동방의 여러 문제들을 해결하는 텍스트 RPG 게임입니다.

## 줄거리
가입 후 아그둥이를 신청한 주인공! 아그둥이 활동을 위해 동방에 들어가게 되는데...

## 등장 적
| 레벨 | 이름 |
|------|------|
| Lv1 | 오늘따라 유독 심한 동방 냄새 |
| Lv2 | 먹고 나서 안치운 커피와 음료수 |
| Lv3 | 신경 쓰지 않아 덜 닫힌 동방 문 |
   
## 전투 시스템
전투는 턴제 방식으로 진행됩니다. 
각 스킬은
- **1. 공격** - 크리티컬 30% 확률로 데미지 2배
- **2. 집중 공격** - 다음 공격 +30 데미지
- **3. 방어** - 이번 턴 피해 0 (연속 사용 불가)

적의 공격은 위 3가지 중 랜덤으로 결정됩니다.
캐릭터의 HP가 0이 되면 전투가 종료되고, 주인공은 여러 존재들의 가호로 다시 도전할 수 있습니다!
